# 💳 Credit Wise Loan Prediction

A Machine Learning project that predicts whether a loan should be approved or rejected based on applicant information. This project helps financial institutions reduce risk and make faster loan approval decisions using data-driven insights.

---

## 📌 Project Overview

Loan approval is a critical process in banking and financial institutions. This project builds a machine learning model that analyzes applicant details such as income, credit history, employment status, and other factors to predict loan eligibility.

The goal is to automate and improve the loan approval process using predictive analytics.

---
🔗 Live App: https://credit-wise-loan-prediction.streamlit.app/

## 🚀 Key Features

### ✅ Interactive Web Application
User-friendly interface built with Streamlit that allows users to input applicant details and instantly receive loan approval predictions.

### 🧠 Machine Learning–Based Prediction
Uses Logistic Regression to classify whether a loan application is **Approved** or **Rejected** based on applicant information.

### 🧹 Robust Data Preprocessing
Comprehensive preprocessing pipeline including:
- Handling missing values
- Encoding categorical variables using One-Hot Encoding
- Feature scaling for improved model performance

### 🛡️ Reliable Prediction Pipeline
Ensures production-safe inference with:
- Proper feature alignment
- Handling unseen categories during prediction
- Consistent preprocessing during training and inference

### 📊 Data Exploration & Visualization
Provides dataset preview and insights using:
- Pandas
- Matplotlib
- Seaborn

### ⚡ Real-Time Loan Prediction
Users can input applicant information and receive **instant loan eligibility predictions** through the web interface.

### ☁️ Cloud Deployment
Application deployed online using Streamlit Cloud for easy access without local installation.
